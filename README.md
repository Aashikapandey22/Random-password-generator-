# Random Password Generator (C++)

## Overview
This project is a Random Password Generator built using C++.  
It generates secure passwords using a combination of uppercase letters, lowercase letters, numbers, and special symbols.

The user can choose the desired password length, and the program generates a random password accordingly.

---

## Features
- User-defined password length
- Random password generation
- Combination of characters:
  - Uppercase letters (A–Z)
  - Lowercase letters (a–z)
  - Numbers (0–9)
  - Special symbols (@ # $ % & *)
- Menu-driven program

---

## Requirements
- C++ compiler (g++ or clang++)
- VS Code or any C++ IDE

---

## How to Run

### 1. Compile the program
```bash
g++ random.cpp -o random
```

### 2. Run the program
```bash
./random
```
